# 📝 Task Manager Frontend

A modern React-based frontend for a full-stack task management application with authentication and password reset functionality.



## 🔗 Live Demo

🌐 https://task-manager-frontend-lemon-ten.vercel.app



## ✨ Features

* 🔐 User Authentication (Login / Register)
* 🔑 Forgot Password & Reset Flow
* 📋 Task Management (Add, View,edit, Delete)
* ⚡ Fast and responsive UI
* 🔄 API integration with backend
* 🚀 Deployed on Vercel



## 🛠️ Tech Stack

* React
* Vite
* Tailwind CSS
* Axios
* React Router DOM



## ⚙️ Environment Variables

Create a `.env` file:


VITE_API_URL=https://auth-backend-3kx0.onrender.com




## 📦 Installation


git clone https://github.com/your-username/TaskManager-frontend.git
cd TaskManager-frontend
npm install
npm run dev




## 🌐 Backend API

This frontend connects to:
